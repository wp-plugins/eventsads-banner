= Description =

This plugin creates one or more Banner Flag for advertising, events, communications, etc.; You can add images, video, take over the title,
subtitle, description and links.

Can be set in open throughout the site or just the home page, or closed throughout the site.

you can pre-set the date of publication and removal for scheduled events.

you can also set a policy (ID / data) for scrolling / banner rotation.

This plugin supports multilingual.

= Settings Banner =

Is possible customize the general settings or individually for each banner.

Among the settings is possible to customize: text formatting, paragraph, background's color/s, position in the window, position (vertical / horizontal) and
banner size, form of the label.

== Installation ==

= Automatic installation =

1. Log into your WordPress admin
2. Click __Plugins__
3. Click __Add New__
4. Search for __Banner Ads - BeB__
5. Click __Install Now__ under "Banner Ads - BeB"
6. Activate the plugin

= Manual installation =

1. Download the plugin
2. Extract the contents of the zip file
3. Upload the contents of the zip file to the wp-content/plugins/ folder of your WordPress installation
4. Then activate the Plugin from Plugins page.

= Requirements =

* PHP 5.2.4+ (same as WordPress 3.5)
* MySQL 5.0+ (same as WordPress 3.5)
* WordPress: 3.5+

= How to create your first Banner ? =

1. After installtion and activation of the plugin;
2. Click on "Events / ADS Banner" -> "Add Banner";
3. The data required are "Banner's Name" (or ID) and the "Title" to be inserted in the banner - everything else is optional;
4. Clicking on "Save" you have created the banner.
